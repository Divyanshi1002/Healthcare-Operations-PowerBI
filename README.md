# Healthcare-Operations-PowerBI

## Project Overview

This project presents a visual analytics dashboard built using the **Maven Hospital Patient Records** dataset, focusing on patient demographics, payer coverage, encounter patterns, and healthcare claims trends.  
The goal of the dashboard is to provide **clear, decision-ready insights** for healthcare operations and business analysis.
